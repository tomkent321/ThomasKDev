Thomas K's Bio

Hello! I am Tom K. Hurd, Thomas K Developers lead (and for now, only) developer. Hey, even Bill Gates had to start somewhere! I am enthusiatically looking for a place to contribute in the development world!

Background

After successful careers in the airline, large retail management and financial worlds, I decided to reinvent myself as a competent, excited and happy full-stack web developer: a career that is a perfect fit for the work ethic, technical skills and people skills I learned in those industries.

Built on the foundation of an intensive code development program at the University of Denver along with a systematic process to hone those skills through a year of intensive follow-up study and practice, I offer development skills that are a step beyond just the basics.

Skills Summary

Continuous learning and learning quickly are my most valuable skills. I can learn what I need in a hurry and apply it to real-world situations immediately. That's how I got a pilot's license and a stock broker's license, a real-estate license and learned REACT. That's how I'm learning Typescript and php/Laravel even as we speak.

Beyond that, I have specialized in mastering the skills and languages needed to be that elusive true full stack web developer. This includes a special focus on SQL and Mongo databases, Node.js, Express and React. I’ve developed React programs with internal state, Redux and contextual APIs and know which work best in various situations. I’ve also dug deeply into all the versions of Javascript and know when to hoist and when not to hoist and took the time to learn how Javascript works behind the scenes.

Finally, we all know it is the user who determines if our efforts worked. I’ve studied and practiced advanced css and scss coding and file organization to be sure that great coding is presented in a great UI. Many years of business experience with the Adobe Creative Suite greatly enhances the capability for developing compelling UI’s.

Developer Connector

This project features a REACT UI with Redux state management. New users are created and users are logged in with Jason Web Token validation. Used bootstrap for CSS.

The back end is driven by node.js and express connected to a Mongo database residing on the Mongo Atlas server. Data is validated on the back end by npm validator. Passwords are encrypted with npm bcrypt.

Deployed on Heroku, so it loads a little slow.
